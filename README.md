# Интерактивное Эмоциональное Письмо

Красивая одностраничная веб-страница с плавными анимациями, созданная для искреннего общения без давления.

## Возможности

- **Плавные scroll-анимации** - каждый блок появляется с эффектами fade-in, blur и подъёма
- **Glassmorphism эффекты** - полупрозрачные карточки с размытием фона
- **Плавающие частицы** - деликатная анимация светящихся частиц на фоне
- **Музыкальный плеер** - возможность включить нежную фоновую музыку (по клику)
- **Анимация букета цветов** - финальная интерактивная сцена с распускающимся букетом
- **Легко редактируемый контент** - весь текст в одном массиве

## Быстрый старт

### 1. Установка зависимостей

```bash
npm install
```

### 2. Запуск в режиме разработки

```bash
npm run dev
```

Откройте [http://localhost:5173](http://localhost:5173) в браузере.

### 3. Сборка для продакшена

```bash
npm run build
```

Готовые файлы будут в папке `dist/`.

### 4. Предпросмотр продакшен-сборки

```bash
npm run preview
```

## Как редактировать содержимое

### Изменить текст сцен

Открой файл `src/App.jsx` и найди массив `scenes`:

```javascript
const scenes = [
  {
    her: "её фраза...",
    me: "мой ответ..."
  },
  // Добавь больше сцен здесь
]
```

Просто измени текст или добавь новые объекты в массив!

### Добавить свою музыку

1. Положи свой аудио-файл в папку `public/` с названием `music.mp3`
2. Или измени путь в файле `src/components/MusicPlayer.jsx` (строка с `<source src="/music.mp3"`)

### Добавить звук колокольчика

Положи аудио-файл `bell.mp3` в папку `public/` для воспроизведения при нажатии на кнопку "Забрать цветы".

## Деплой

Ты можешь задеплоить этот сайт на:

### Vercel

```bash
npm install -g vercel
vercel
```

### Netlify

1. Собери проект: `npm run build`
2. Перетащи папку `dist/` на [Netlify Drop](https://app.netlify.com/drop)

### GitHub Pages

1. Установи `gh-pages`:
```bash
npm install -g gh-pages
```

2. Добавь в `vite.config.js`:
```javascript
export default defineConfig({
  base: '/название-репозитория/',
  plugins: [react()],
})
```

3. Собери и задеплой:
```bash
npm run build
gh-pages -d dist
```
